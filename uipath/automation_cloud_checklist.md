# UiPath Automation Cloud Checklist

## Tenant and access
- Verify Automation Cloud tenant access
- Verify Maestro BPMN permissions
- Verify Robot runtime availability

## Integration setup
- Configure API endpoints for compliance-risk agent and prioritization agent
- Configure secure credentials and connection assets
- Validate network access and auth from UiPath to agent endpoints

## Process deployment
- Publish BPMN process and dependencies
- Configure queues, triggers, and environments
- Add user tasks for procurement lead and escalation owner

## Observability
- Enable logs and process analytics
- Confirm run history and error traces are visible
- Capture screenshots for submission

## Demo readiness
- Prepare one happy path emergency requisition
- Prepare one high-risk requisition needing approval
- Prepare one blocked requisition needing escalation
