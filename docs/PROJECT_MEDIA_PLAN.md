## Image quality checklist
- Use PNG when possible
- Keep text legible at 1366x768 minimum
- Remove or blur sensitive identifiers
- Maintain 3:2 ratio (e.g., 1500x1000)
- Avoid duplicated screenshots with minor UI differences

## Video demo link requirements
- Platform: YouTube, Vimeo, or Youku
- Duration: 5 minutes max
- Must show real system running (not slides only)
