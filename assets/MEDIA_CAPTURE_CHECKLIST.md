# Media Capture Checklist

Use this quick checklist while capturing assets.

## Before capture
- [ ] Close unrelated tabs/apps
- [ ] Increase zoom for readability
- [ ] Prepare happy path and exception test data
- [ ] Hide credentials, tenant IDs, and private names

## Capture order
- [ ] BPMN flow canvas
- [ ] Architecture diagram
- [ ] Intake payload
- [ ] Risk scoring output
- [ ] Human approval task
- [ ] PO creation outcome
- [ ] Prioritization output
- [ ] Run history/log trace

## Export and naming
- [ ] Save PNG files in 3:2 ratio
- [ ] Use naming: crisisprocure-<topic>-NN.png
- [ ] Confirm each image is under 5 MB

## Final submission
- [ ] Upload 8 to 10 images to Devpost gallery
- [ ] Add demo video link
- [ ] Verify first image is the strongest proof shot
